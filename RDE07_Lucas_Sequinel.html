Branches

São ramificações de um projeto de software ou documento
e tem como objetivo o desenvolvimento paralelo de:

  • Novas funcionalidades
  • Correção de problemas
  • Alteração de código
Branch Master: Braço principal;
Branch Secundária: Braço secundário;
As braches secundárias servem para fazer alterações no código sem alterar o original
podendo ser utilizadas para testes e usos cooperativos de programadores.
Para adicionar uma atualização da Branch secundária se utiliza o Merge
